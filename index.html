About Me

My name is Bar Abudi. My journey with technology started when I received my first PC at the age of 4. Since then I enjoyed solving problems using technology. I share my passion by teaching others how to develop applications, websites and games using various programming languages like Python, JavaScript, Java and more!

I created this website to practice Python, creating Docker images, deploying on cloud VMs (Oracle & Aws), and serving websites using Flask. The source files are in a public GitHub repository
